# Rolodex Contact Manager

## Project Overview
This is a C# console-based Rolodex application that allows users to manage contacts using object-oriented programming principles and SQLite database storage.

## Features
- Create, view, update, and delete contacts
- Multiple contact types (Business, Family, Friend)
- SQLite database persistence
- Menu-driven console interface
- Demonstrates inheritance, abstraction, polymorphism, and composition

## Technologies Used
- C#
- .NET 6
- SQLite
- Git/GitHub

## How to Run
1. Clone the repository
2. Open the project in Visual Studio or VS Code
3. Run the application using `dotnet run`
